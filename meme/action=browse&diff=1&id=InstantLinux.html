    == * General ==
     
    <b>[http://www.knoppix.net/ KNOPPIX]</b> is a bootable CD with a large collection of free software, automatic hardware detection, and support for many graphics cards, sound cards, SCSI and USB devices and other peripherals. It's can be used as a Linux demo, educational CD, rescue system, or adapted and used as a platform for commercial software product demos. It is not necessary to install anything on a hard disk. Due to on-the-fly decompression, the CD can have up to 2 GB of executable software installed on it.
     
    * [http://www.pjls16812.pwp.blueyonder.co.uk/knowing-knoppix/index.html Knowing Knoppix] is the first guide to Knoppix for the complete beginner.
    * [http://www.oreilly.com/catalog/knoppixhks/chapter/index.html 5 Free Knoppix Hacks]
     
    <b>[http://www.ubuntulinux.org Ubuntu] / [http://www.kubuntu.org Kubuntu]</b> is a complete Debian Linux-based operating system, freely available with both community and professional support. [http://www.ubuntulinux.org/download Download] or check the [http://www.ubuntulinux.org/shipit free cd offer].
     
    == * Media Production ==
     
    [http://dynebolic.dyne.org/ Dyne:bolic GNU/Linux] is a live bootable distribution, an operating system which works directly from the CD without the need to install or change anything on the hard disk. Shaped by the needs of media activists, artists and creatives. With it one can manipulate and broadcast both sound and video with tools to record, edit, encode and stream. It is user-friendly, recognizes your hardware devices (sound, video, firewire, and USB) and offers a vast range of software for multimedia production, streaming, 3D modeling, photo, peer-to-peer filesharing, web browsing and publishing, word processing, email, encryption, and networking. It also includes games and a world navigator. It does automatic clustering, joining the CPU power between any other dyne:bolic on the local network, and works on modded XBOX consoles as well.
     
    [http://lamppix.tinowagner.com LAMPPIX] is a Live CD with the web server Apache, the database <no>MySQL</no> and the programming languages PHP and [[Perl]]. It's based on Klaus Knopper's KNOPPIX and Damn Small Linux. LAMPPIX allows you to easily run your dynamic web pages off a CD-ROM.
     
    [http://www.frozentech.com/content/livecd.php?pick=All&showonly=Media+Production&sort=&sm=1 More...]
     
    == * Media Players ==
     
    [http://geexbox.org/en/index.html GeeXboX] is a standalone media player Linux distribution, similar to MoviX. It's a small bootable CD which allows you to play your favorites video and audio files. It is based on MPlayer, and can be used on every x86 computer. It supports TV-out for various cards such as S3, nVidia, and ATI. It also supports network and is able to play medias on a Windows/Samba share. It also supports TV tuners and WiFi cards.
     
    [http://movix.sourceforge.net MoviX] is a small-sized (~50MB) Linux "live CD" distro containing all the software to boot from any media you like and start automagically X-window and within it the GUI version of MPlayer, which can play DVDs, <no>VCDs</no>, <no>DivX</no>, <no>AudioCDs</no> and many more.
     
    [http://womp.sourceforge.net/ WOMP!] is a micro linux distribution focussed on multimedia. Basically, it takes only 13 to 30 MB depending on the selected options on a bootable CD, and allow playing a wide range of multimedia files (video/audio/image) without installing any software on the computer's hard drive.
     
    [http://www.frozentech.com/content/livecd.php?pick=All&showonly=Home+Entertainment&sort=&sm=1 More...]
     
    == * Bioinformatics ==
     
    [http://bioknoppix.hpcf.upr.edu/ Bioknoppix] is a customized distribution of Knoppix Linux Live CD with open source applications targeted for the molecular biologist.
     
    [http://public.vigyaancd.planetmirror.com/ Vigyaan] is an electronic workbench for bioinformatics, computational biology and computational chemistry. It has been designed to meet the needs of both beginners and experts. <no>VigyaanCD</no> is a live Linux CD based on KNOPPIX v3.3 containing all the required software to boot the computer with ready to use modeling software.
     
    [http://www.dnalinux.com DNALinux] is a Slackware Linux distribution with bioinformatics software pre-installed. [DNA] and protein sequence manipulation software is included.
     
    [http://cdmedicpacsweb.sourceforge.net/cdmedic_en.html CDMEDIC] is a Knoppix variant appropriate for Radiologists, Nuclear medicine and Radiotherapy Physicians and MD, Neurologists and Neurosurgeons and also other branches related to Medical imaging as Physics and Informatica.

    [http://www.sourcepole.com/sources/software/gis-knoppix/index.xml GIS-Knoppix] is a bootable Linux CD with pre-installed GIS software.

    == * Wireless ISP ==
     
    [http://www.publicip.net/ Public IP] provides hotspot operators with an extremely simple, highly versatile, FREE, open source hotspot solution. Public IP's <no>ZoneCD</no> is freely distributed software that has been created to help implement safe, free, <no>WiFi</no> hotspots.

    == * Small ==
     
    [http://www.damnsmalllinux.org Damn Small Linux] is a very versatile 50MB mini desktop oriented Linux distribution.
     
    [http://featherlinux.berlios.de Feather Linux] is a Linux distribution which runs completely off a CD or a USB pendrive and takes up under 115Mb of space. It is a Knoppix remaster and tries to include software which most people would use every day on their desktop.
     
    [http://slax.linux-live.org/ SLAX] is a small bootable CD based on the Slackware Linux distribution.
