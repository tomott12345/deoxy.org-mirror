    '''The Human Biocomputer''' coined by [Deoxy:lilly.htm John C. Lilly], refers literally to the "Videx:hardware" of the human Videx:anatomy. This would include the Videx:brain, internal organs, and other human organ systems such as Videx:Cardiovascular, Videx:Digestive, Videx:Endocrine, Videx:Immune, Integumentary, Videx:Lymphatic, Muscular, Videx:Nervous, Videx:Reproductive, Videx:Respiratory, Skeletal, and Videx:Urinary systems. The Videx:biocomputer has stored [Deoxy:8brains.htm program] properties, and [Deoxy:h_lilly.htm self-metaprogramming] properties, with limits determinable and to be determined.

    == * Definitions ==
    <blockquote>
    ;'''Mind''':Mind, which is defined as the sum total of all the Programs and Metaprograms (and even supraself metaprograms) of a human biocomputer. This is the software and is looked at as the opposite of the hardware.
     
    ;'''The Brain''':The brain is defined as the visible, palpable living set of structures to be included in the human biocomputer.
     
    http://deoxy.org/vi/yGTiGZsBB8A.jpg|right12
    ;'''Stored programs''':A stored program is defined as a set of instructions which are placed in memory storage of the biocomputer, and which control the biocomputer when orders are given for that program to be activated. These programs can be activated by the same biocomputer, another biocomputer, or a situation outside of the biocomputer.
    </blockquote>
